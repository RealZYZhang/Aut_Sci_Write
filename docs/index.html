技能详情

每个技能独立可用，也可组合成完整工作流

🔍
sci-search
文献检索与期刊指标
同时搜索 arXiv 和 PubMed，自动附加 JCR 分区、影响因子，高亮 Nature/Science 等顶刊。
搜索文献 查文献 影响因子 find papers
📄
sci-extract
论文核心内容提取
从 PDF 中提取研究问题、方法、关键结论、实验参数和数值对比，超越简单摘要。
分析论文 提取结论 论文要点 analyze paper
🖼️
sci-figure
论文图表智能提取
自动检测 PDF 中的图表位置，高精度裁剪（600 DPI），支持复合图拆分为独立子图。
提取图片 截取子图 extract figure
✍️
sci-review
综述写作与审稿回复
基于 4 阶段系统综述结构，辅助撰写文献综述；专业审稿回复模板，对标 NeurIPS/ICLR 标准。
文献综述 审稿回复 论文润色 rebuttal
📚
sci-zotero
Zotero 文献库集成
同步 Zotero 远程文献库，支持通过 DOI/ISBN/PMID 添加引用，自动获取开放获取 PDF。
同步文献 Zotero add citation
📊
sci-ppt
学术 PPT 自动生成
从论文 PDF 或结构化文本一键生成专业学术 PPT，支持 LaTeX 公式渲染、图片嵌入、中英双语布局。
生成PPT 组会汇报 答辩PPT 论文转PPT

快速安装

三步完成配置，立即开始使用

Step 1 — 克隆仓库
git clone https://github.com/ShZhao27208/Aut_Sci_Write.git
cd Aut_Sci_Write
pip install -r requirements.txt
Step 2 — 配置环境变量
# Zotero 集成（可选）
export ZOTERO_API_KEY=your_api_key
export ZOTERO_USER_ID=your_user_id

# PPT 生成（选择其一）
export ANTHROPIC_API_KEY=sk-ant-...
export MOONSHOT_API_KEY=sk-...
Step 3 — 在 Claude Code 中注册
# 在 Claude Code 会话中执行：
/oh-my-claudecode:skill setup
